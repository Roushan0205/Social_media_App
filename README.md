# Social_media_App

An interactive **Social Media App** that allows users to upload an image, apply various filters (like Grayscale, Sepia, Vintage, Cool, Warm, etc.), adjust filter intensity, preview results in real-time, and download the final edited image.

---

## 🚀 Features

- 🖼️ Upload any image from your device  
- 🎛️ Apply multiple CSS-based filters:
  - Grayscale  
  - Sepia  
  - Invert  
  - Blur  
  - Brightness  
  - Contrast  
  - Saturate  
  - Hue Rotate  
  - Opacity  
  - Vintage, Cool, and Warm presets  
- ⚡ Adjust intensity with a smooth slider  
- 💾 Download the filtered image instantly  
- 🧹 Reset filters anytime  
- 📱 Fully responsive layout  

---

## 🧰 Tech Stack

- **React.js** (Vite)
- **CSS3**
- **JavaScript (ES6+)**

## ⚙️ How to Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/<your-username>/SOCIAL_MEDIA_APP.git
   cd SOCIAL_MEDIA_APP
2. **Install dependencies**
    cd frontend 
    npm install
3. **Start the development server**
    npm run dev
4. **Open your browser at**
   👉 http://localhost:5173
